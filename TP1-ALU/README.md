# Trabajo Practico #1 - ALU 

## Objetivo
Implementar en FPGA una ALU (Arithmetic Logic Unit).

La misma debe ser parametrizable.

Su desarrollo debe ser validado por medio de Test Bench.

Debe realizar las siguientes operaciones:
| Operaciones | Codigo |
|:-----:|:-----:|
| ADD | 100000 |
| SUB | 100010 |
| AND | 100100 |
| OR | 100101 |
| XOR | 100110 |
| SRA | 000011 |
| SRL | 000010 |
| NOR | 100111 |
